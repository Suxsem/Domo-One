PR = photoresistor (with long pins)

R1, R2 = 10Komh
R3 = 100omh
R4, R5 = 150omh

L1, L2, L3 = common cathode RGB leds (the ground is on the middle-upper pin)

dht = female pin header for dht22 temperature and humidity sensor

domo1, domo2 = long male pin headers for the domoone-1 pcb, soldered at the back of the pcb
